## 🚀 What is TOC Generator?

TOC Generator is a powerful Figma plugin that automatically creates professional **Table of Contents** from your slides. Perfect for presentations, documentation, and any project that needs organized navigation.

### ✨ Key Features

- **🎯 Smart Slide Detection** - Automatically finds and numbers your slides
- **🎨 Beautiful TOC Layouts** - Multi-column, responsive designs
- **⚡ Real-time Updates** - TOC updates when you rename slides
- **🎨 Customizable Styling** - Fonts, colors, spacing, and more
- **📱 Responsive Design** - Works with any frame size
- **🔄 Multiple Ordering Options** - Z-order, vertical, horizontal layouts

---

## 🎯 Quick Start Guide

### Step 1: Add Slide Numbers
1. Open your Figma project
2. Run the **TOC Generator Plugin**
3. Click **"Add Slide Numbers"** to number your slides
4. Use the direction buttons to control slide ordering:
   - **Z**: Left-to-right, top-to-bottom
   - **Vertical**: Top-to-bottom
   - **Horizontal**: Left-to-right
   - **Z-Flipped**: Right-to-left, top-to-bottom

### Step 2: Generate TOC
1. **Select a frame** in your document (this is where the TOC will be created)
2. Click **"Generate TOC"** 
3. Your beautiful table of contents appears! ✨

### Step 3: Customize (Optional)
- **TOC Style**: Customize fonts, colors, and text styling
- **TOC Layout**: Adjust spacing, columns, and positioning
- **Number Style**: Change number appearance and position

### 📝 Numbering Section (Top)
**Purpose**: Add numbers to your slides and control their order

**Buttons Explained**:
- **🔢 Add Numbers**: Numbers all detected slides
- **🔄 Refresh**: Reload slides from Figma
- **🔧 Fix**: Renumber slides sequentially (removes gaps)
- **❌ Remove**: Delete all slide numbers
- **⚙️ Settings**: Customize number appearance

**Direction Buttons**:
- **Z**: Most common - reads slides like a book
- **Vertical**: Top to bottom ordering
- **Horizontal**: Left to right ordering
- **Z-Flipped**: Right to left reading

### 📋 TOC Generator Section (Bottom)
**Purpose**: Create beautiful table of contents

**Features**:
- **Smart Detection**: Only numbered slides appear in TOC
- **Auto-layout**: Professional multi-column designs
- **Live Updates**: TOC changes when you rename slides
- **Custom Styling**: Full control over appearance

## 💡 Pro Tips

### 🎯 Best Practices
1. **Use Clear Slide Names**: "Introduction" vs "Slide 1"
2. **Consistent Naming**: "Section 1: Overview" format
3. **Logical Ordering**: Arrange slides in reading order
4. **Test Different Directions**: Try Z-order first, then others

### 🔧 Troubleshooting
- **No Slides Detected**: Make sure frames are visible and unlocked
- **Wrong Order**: Try different direction buttons
- **Numbers Missing**: Click "Add Slide Numbers" first
- **TOC Not Updating**: Rename slides and regenerate

### ⚡ Advanced Features
- **Nested Slides**: Create parent-child relationships
- **Custom Numbering**: Start from specific slides
- **Batch Operations**: Update multiple slides at once
- **Style Presets**: Save and reuse custom styles
